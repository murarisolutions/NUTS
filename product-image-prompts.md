| **Bottom Gusset Fold** | 165–200mm | 200–240mm | 245–290mm | Hidden when standing |

### File Setup for Designer/Printer

| Parameter | Value |
|-----------|-------|
| **Resolution** | 300 DPI (minimum) |
| **Color Mode** | CMYK |
| **File Format** | AI / PDF / PSD |
| **Bleed** | 3mm on all sides |
| **Safe Zone** | 5mm inside trim line |
| **Font Minimum** | 6pt for legal text, 12pt for product name |
| **Barcode Size** | Minimum 25mm x 20mm |
| **Veg Mark** | 12mm x 12mm green square with green circle |
